# Правила обработки изображений

Изображения обрабатываются последовательным применением **правил** (rules). Каждое правило — это пара `ключ => конфиг`, где ключ — название операции, конфиг — её параметры.

```php
$rules = [
    'crop'   => ['width' => 200, 'height' => 200],
    'grayscale' => [],
];
```

Правила применяются в порядке объявления. Результат предыдущего правила передаётся в следующее.

---

## resize

Вписать изображение в прямоугольник `width × height` с сохранением пропорций. Итоговый размер может быть меньше указанного.

```php
'resize' => ['width' => 300, 'height' => 200]
```

## crop

Центрированное кадрирование до точных `width × height`.

```php
'crop' => ['width' => 150, 'height' => 150]
```

## rotate

Поворот на угол, кратный 90°.

```php
'rotate' => ['angle' => 90]   // 90 | 180 | 270
```

## grayscale

Преобразование в оттенки серого. Без параметров.

```php
'grayscale' => []
```

## watermark

Наложение PNG-изображения (watermark) с прозрачностью.

```php
'watermark' => [
    'image'    => __DIR__ . '/public/assets/wm.png',  // путь к PNG
    'width'    => 100,    // опционально: ширина watermark
    'height'   => null,   // опционально: высота (пропорции, если только width)
    'position' => 'SE',   // NW | NE | SW | SE
    'gap'      => 10,     // отступ от края в px
]
```

Если `width`/`height` не указаны — используется оригинальный размер файла.  
Если указан только `width` — высота сохраняет пропорции (и наоборот).

---

## Профили

Профили в конфиге сервиса — это именованные наборы правил:

```php
'profiles' => [
    'thumb' => [
        'crop'   => ['width' => 150, 'height' => 150],
        'grayscale' => [],
    ],
    'preview' => [
        'resize'  => ['width' => 600, 'height' => 400],
    ],
    'watermarked' => [
        'resize' => ['width' => 1200, 'height' => 900],
    ],
],
```

Запрос: `?profile=thumb` применит оба правила (crop → grayscale).

---

## Query-параметры

При запросе без `profile` правила формируются из параметров:

```
?width=300&height=200&mode=crop
```

Преобразуется в:

```php
['crop' => ['width' => 300, 'height' => 200]]
```

Параметр `mode` может быть `resize` (по умолчанию) или `crop`.

---

## Очерёдность имеет значение

```php
// crop → grayscale
['crop' => ['width' => 100, 'height' => 100], 'grayscale' => []]

// grayscale → crop (сначала обесцветить, потом кадрировать)
['grayscale' => [], 'crop' => ['width' => 100, 'height' => 100]]
```

Результат может отличаться.
